# A.snap

This is a premium short video sharing application built with Next.js, Firebase, and Tailwind CSS.

## Features

- **Short Video Feed:** Smooth scrolling video feed with auto-play and mute controls.
- **Visual Sharing:** Share posts directly with friends in real-time chats with rich video previews.
- **Instagram-style Chat:** Real-time messaging with long-press to delete (unsend) functionality.
- **Advanced Search:** Find users by username with case-insensitive search.
- **Monetization Ready:** Integrated sponsored post system for advertisements.
- **Profile Management:** Complete profile editing and video management (delete posts via three-dots menu).

To get started, run `npm run dev`.
